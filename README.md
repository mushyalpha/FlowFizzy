# Power Node - Smart Energy Controller 
## Overview

Power Node Product is a **real-time centralized power controller** implemented in C++ on Raspberry Pi, designed to intelligently manage and prioritize power consumption across multiple energy sources such as batteries.
Inspired by Virtual Power Plant (VPP) concepts, the system operates at the edge, dynamically allocating available power to connected loads based on source availability, state-of-charge, and configurable priority policies.
The platform targets embedded energy management use cases requiring deterministic behavior, low-latency decision making, and high reliability.

## Key Features

- Real-time power dispatch and load prioritization
- Multi-source energy management (e.g., batteries, external supplies)
- Policy-based load control (critical / non-critical / shed-able)
- Modular C++ architecture for embedded deployment
- Designed for Raspberry Pi with real-time constraints
- Extensible framework for future VPP-style coordination

## Use Cases

- Edge energy management systems
- Battery-backed power controllers
- Smart microgrids
- Embedded VPP nodes
